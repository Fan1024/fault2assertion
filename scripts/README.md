# Scripts

This directory contains repository-level automation.

Current script:

- `inspect_crc32.sh`
  - lists the imported CRC32 files;
  - identifies the entry point and CRC functions;
  - identifies file-I/O, command-line, library, and operating-system
    dependencies;
  - reports available input-data files and their sizes.

Build, simulation, fault-injection, and campaign scripts will be added
only when their required interfaces are known.
